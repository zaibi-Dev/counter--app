# Counter App

A simple counter application built using HTML, CSS, and JavaScript.

## Features

- Increment counter
- Decrement counter
- Reset counter
- Clean UI
- Beginner friendly project

## Technologies Used

- HTML
- CSS
- JavaScript

## Project Structure

counter--app/
│
├── index.html
├── style.css
├── script.js
├── background image.jpeg
└── README.md

## How to Run

1. Download or clone the repository
2. Open index.html in any web browser
3. Use +, -, and Reset buttons to control the counter

## Learning Purpose

This project is created for learning JavaScript DOM manipulation and basic frontend development.
